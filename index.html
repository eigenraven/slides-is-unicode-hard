Solution?

  • Strings != Arrays of characters
  • Unicode algorithms

  • Don't assume:
    • Single byte characters
    • 1-to-1 code point-glyph mapping
    • The length of a string being the number of characters
    • \n or \r\n for newlines
    • Strings being equal if bytes are equal
